Step 1: Groq API Key
Groq powers all the AI features. It's completely free — no credit card needed. Works from UAE.
1. Go to console.groq.com/keys
2. Sign up (free) → click "Create API Key"
3. Copy the key (starts with gsk_)
4. Paste it below
📱
Step 2: WhatsApp Notifications
1. Save +34 644 91 71 62 as a contact in WhatsApp
2. Send this EXACT message: I allow callmebot to send me messages
3. You'll receive your API key back in a few seconds
4. Enter your phone and key below
📧
Step 3: Email Notifications
1. Enable 2-Factor Authentication on your Google account
2. Go to myaccount.google.com/apppasswords
3. Create a new app password → select "Mail"
4. Copy the 16-character password below
🚀
Step 4: Deploy Backend to Render
1. Push the backend/ folder to a GitHub repo
2. Go to render.com → New → Web Service
3. Connect your GitHub repo
4. Build: npm install | Start: npm start | Instance: Free
5. Add all environment variables (from .env.example)
6. Copy your Render URL and paste below
7. Set up cron-job.org (see README.md)
